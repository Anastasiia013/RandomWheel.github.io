# Проект "Колесо Имен и Создание Команд"

Этот проект предоставляет систему, где пользователи могут создать колесо имен, вводя имена и выбирая победителя в конце. Кроме того, он создает случайные команды по 5 человек, назначая капитана и программиста каждой команде.

## Особенности

### Колесо Имен:
- Пользователи добавляют имена в колесо.
- Колесо случайным образом выбирает имя, чтобы объявить победителя.

### Создание Команд:
- Случайным образом создает 4 команды из введенных имен.
- Назначает капитана и программиста каждой команде случайным образом.

## Инструкции по использованию

### 1. Колесо Имен:
- Добавьте имена в систему.
- Нажмите кнопку "Вращать колесо".
- Имя победителя будет отображено на экране.

### 2. Создание Команд:
- Команды автоматически создаются из списка введенных имен.
- Каждой команде случайным образом назначаются капитан и программист.
- Список команд и их роли будет отображен.

## Архитектура Проекта

- **Фронтенд**: Удобный интерфейс, анимация вращения колеса и детали команд.
- **Бэкенд**: Алгоритмы для случайного выбора имен и создания команд.
- **База данных**: Временно хранит имена пользователей и информацию о командах.
